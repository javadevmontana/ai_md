# 유상·무상 작업 정의 기준

> **전제조건:** 유지보수 계약이 체결되어 있어야 지원 가능

---

## 무상 작업

| 구분 | 내용 |
|------|------|
| **패치 작업** | 고객사 패치 작업은 무상 지원 |
| **장애 대응** | 유지보수 계약상 무상 지원. 단, 엔지니어 섭외에 시간이 소요되며 즉시 지원이 어려울 수 있음 |
| **고객사 자체 작업 중 장애** | 고객사 자체 작업 중 장애 발생 시 무상 지원 |
| **일반 문의** | 무상 |
| **보안 관련 교육 방문** | 무상 |
| **기타 이슈 협의 방문** | 무상 |

---

## 유상 작업

| 구분 | 내용 |
|------|------|
| **정기 PM / 서버이전 / 건물정전 등** | 작업시간 동안 방문 또는 유선 대기 시 유상 ※ 1MD 이하 소규모 작업의 경우, 고객사와의 관계를 고려하여 탄력적으로 무상 처리 권고 |
| **감사용 데이터 추출 등** | 제품 기능과 무관한 별도 업무는 유상 ※ 1MD 이하 소규모 작업의 경우, 고객사와의 관계를 고려하여 탄력적으로 무상 처리 권고 |

### 유상 작업 공수 산정 기준

| 조건 | 산정 기준 |
|------|-----------|
| **유선 대기** | 실제 작업 공수(MH) 차감 |
| **방문 지원** | 기본 **1MD** 산정 |
| **휴일 및 주말** | 실제 작업 공수 **2배** 또는 **1MD** 산정 |

> **안내 사항:** 사전에 계획된 작업의 경우, 고객사 요청 시 위 유상 기준을 사전 안내해야 합니다.
>
> 엔지니어를 대기 및 지원받아 빠른 처리를 하는 데에는 그만큼 비용이 필요합니다.
