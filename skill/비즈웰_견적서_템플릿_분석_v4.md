## 11. 견적서 생성 가이드

### 새 견적서 생성 시 필요 정보
1. **고객사명** → B5 셀
2. **견적번호** (YYYY-고객사-유형##) → B4 셀
3. **견적일자** → B6 셀
4. **견적명** (프로젝트명) → B9 셀
5. **구분** (대분류) → B14 셀
6. **내용** (상세 설명) → C14 셀
7. **수량** (M/D) → E14 셀
8. **공급단가** (기본 500,000원) → G14 셀
9. **할인 여부** (Special DC) → 별도 행 추가
10. **부가세 포함 여부** (부가세별도 / 부가세포함) → B10 셀 수정

### 공식 템플릿 사용법
```
1. 기술지원팀_개발_견적서_템플릿.xlsx 파일 복사
2. B4: 견적번호 수정 (예: 2026-ABC회사-D01)
3. B5: 고객사명 수정 (예: ABC회사)
4. B6: 견적일자 수정 (예: 2026.02.04)
5. B9: 견적명 수정 (예: OTP 인증 기능 추가 개발)
6. B14: 구분 수정 (예: 그룹웨어 로그인 OTP 추가)
7. C14: 내용 수정 (예: Web/Mobile OTP 인증 개발)
8. E14: 수량(MD) 수정 (예: 10MD)
9. H14: 공급금액 수정 (예: 5000000)
10. B10: 견적금액 한글 표기 수정
11. H15: 합계 수식 확인 (=SUM(H14))
```
